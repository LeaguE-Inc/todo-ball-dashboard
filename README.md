# ⚽ ボール管理 TODO ダッシュボード

仕事タスクの「ボール（責任）」を可視化して管理するインタラクティブなTODOシステム。

## 🎯 特徴

- **視覚的な色分け**: 優先度とボール所有者で一目瞭然
- **カウントダウン表示**: 期限までの日数をリアルタイム表示
- **進捗率管理**: 各タスクの完了度を可視化
- **ドラッグ&ドロップ**: 直感的なセクション移動
- **自動リマインド**: 毎朝8時に期限超過タスクを自動通知
- **ローカルストレージ**: データはブラウザに自動保存

## 🔧 機能説明

### ダッシュボード (index.html)

- **5つのボード**: 優先度別（高/中/低）、対応待ち、リマインド
- **タスク操作**:
  - ドラッグ&ドロップで移動
  - ボール表示クリックで所有者変更
  - 右クリックで削除
  - 「+」ボタンで新規追加

### 自動リマインド (todo_ball_checker.py)

毎朝8時に自動実行:
1. TODOデータ読込
2. 対応期限をチェック
3. 期限超過タスクを検出
4. リマインドに追加

## 🔐 プライバシー

- すべてのデータはブラウザのLocalStorageに保存
- サーバーに送信されない（完全ローカル管理）
